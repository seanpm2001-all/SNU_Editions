SNU Lite

Removed features

About

SNU Lite is designed to be the lightest-weight variation of SNU. The main branch is heavy, and SNU branch is going to be a lot lighter.

Features that are removed include:

* Built-in user accounts (SeanPatrickMyrickFirefox, Filthy Frank, PiePivotMontier,)

* All snapshots (snapshots will be installable instead)

* All content from the old Legends of the Internet pages

* Repositories of games and software (Facade, Minecraft Python Edition, WacOS)

* Big Buck Bunny full suite

* All snapshot sections from every page

This will make the lite distribution take up over 32 Gigabytes less than the full version.
